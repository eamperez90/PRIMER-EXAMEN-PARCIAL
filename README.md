# Iterativo vs Recursivo

## Qué incluye
- 8 implementaciones: 4 algoritmos (Factorial, Fibonacci, Búsqueda lineal, Burbuja) en versión **iterativa** y **recursiva**.
- Medición de tiempos con `System.nanoTime()` y **promedio de 5 ejecuciones**.
- Exporta resultados a `data/results.csv` para graficar en Excel.

## Requisitos
- Java 21+
- Maven 3.8+

## Cómo ejecutar
En la raíz del proyecto:

```bash
mvn -q clean compile exec:java
```

Al finalizar, se genera/actualiza:
- `data/results.csv`
